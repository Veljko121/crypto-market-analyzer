# Crypto Market Analyzer

## Osnovni podaci
- Ime i prezime: Veljko Nikolić
- Indeks: E2 37/2024
- Predmet: Arhitekture sistema velikih skupova podataka

## O projektu

Zadatak ovog projekta je analiza istorijskih i trenutnih podataka o kriptovalutama. Za istorijski skup podataka izabran je skup podataka [Coinmarketcap historical data](https://www.kaggle.com/datasets/bizzyvinci/coinmarketcap-historical-data), a za dobavljanje trenutnih podataka koristi se [CoinGecko API](https://www.coingecko.com/en/api).
